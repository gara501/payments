Metrics Page

  1. Dashboard

Multi-level Path Example

For deeper paths like /settings/profile, the breadcrumb shows all levels:

  1. Dashboard
  2. Settings
  3. Profile

Hyphenated Path Example

Paths with hyphens like /user-settings are automatically formatted:

  1. Dashboard
  2. User Settings

Breadcrumb Features

  • Automatic Path Detection: Breadcrumbs are generated automatically based on the current URL
  • Smart Formatting: Hyphenated paths are converted to readable labels (e.g., "user-settings" â†’ "User Settings")
  • Clickable Navigation: All breadcrumb items except the current page are clickable links
  • Accessibility: Proper ARIA labels and semantic HTML for screen readers
  • Visual Separators: Chevron icons separate breadcrumb items for clear hierarchy
  • Hidden on Home: Breadcrumbs don't display on the home page to avoid redundancy
